# BSS_PRAVO - statnicove poznamky

Hlavni soubor pro Overleaf je `main.tex`.

Obsah:

- otazky 15-23 z bloku Pravo a politika pro kyberbezpecnost,
- rozsirene studijni poznamky podle presneho zneni podotazek,
- stavova tabulka predpisu a strategii k 16. 6. 2026,
- rychle opakovani pred zkouskou,
- seznam overenych online zdroju.

Material vychazi z prilozeneho PDF `Statnice_B-CS_blok3_otazky_15-23_v4.2.pdf`.
Pro BSS otazky 15-17 byly navic zapracovany dodane starsi poznamky.
U pravnich otazek je pouzit aktualizovany stav pravni upravy, hlavne novy zakon
c. 264/2025 Sb., e-evidence, eIDAS 2.0, CRA a aktualni stav mezinarodnich umluv.

Poznamka: lokalni stroj v teto Codex session nema v PATH nainstalovany LaTeX
kompilator (`pdflatex`, `latexmk`, `xelatex` ani `lualatex`). Projekt je proto
pripraven pro kompilaci v Overleafu.
